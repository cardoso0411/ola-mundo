# ola-mundo

Este projeto é um exemplo simples de uma página web utilizando HTML e CSS. 
O objetivo é demonstrar a criação de uma página básica, normalmente usada como o famoso "Olá, mundo" — um primeiro passo para aprender como construir e estilizar páginas na web.

**Linguagens utilizadas**:
- **HTML:** responsável pela estrutura da página.
- **CSS:** utilizado para definir o estilo e aparência visual.

Ideal para quem está começando a programar e deseja entender os fundamentos do desenvolvimento web front-end.
